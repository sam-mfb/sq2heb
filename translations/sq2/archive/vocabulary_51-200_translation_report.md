## החלטות תרגום מיוחדות

### 1. פעלים - צורות מרובות

בהתאם להנחיות התרגום, **פעלים כוללים 3 צורות עיקריות:**
- **ציווי:** הבט, קח, פתח
- **עתיד:** תראה, תקח, תפתח
- **שם פועל:** לראות, לקחת, לפתוח

**דוגמה:** `attack` (word #54)
```
תקוף, לתקוף, תתקוף, הרוס, להרוס, תהרוס, הרוג, להרוג, תהרוג
```

### 2. שמות עצם - צורה בסיסית בלבד

בהתאם להנחיות, **שמות עצם ניתנים בצורת הבסיס בלבד** (הקומפיילר מייצר prefix אוטומטית):

**דוגמה:** `door` (word #22 מ-Phase 2a)
```
דלת, דלתות  ← רק צורות יחיד/רבים
NOT: הדלת, לדלת, בדלת  ← אלו נוצרים אוטומטית
```

### 3. מילות debug

המילים הבאות **לא תורגמו** כי הן פקודות טכניות:
- `mem`, `pri`, `c`, `cm`, `s`, `sn`, `t`, `var`, `dbg`, `backstage`, `hh`
